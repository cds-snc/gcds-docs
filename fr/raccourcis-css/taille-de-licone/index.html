Taille de l'icône

La classe taille de l'icône définit les propriétés font-size et line-height des icônes. Elle détermine la taille de gcds-icons.

Survol

Raccourci de classe CSS Style appliqué

font-size: var(--gcds-font-sizes-h1);
line-height: var(--gcds-line-heights-h1);
