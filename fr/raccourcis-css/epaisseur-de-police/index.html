Épaisseur de police

La classe poids de police définit la propriété font-weight. Elle détermine à quel point le texte est en gras.

Consulter les directives typographiques de Système de design GC pour choisir le bon poids de police pour votre contexte.

Survol

Raccourci de classe CSS Style appliqué 
font-regular
 
font-weight: var(--gcds-font-weights-regular);
 
font-light
 
font-weight: var(--gcds-font-weights-light);
 
font-medium
 
font-weight: var(--gcds-font-weights-medium);
 
font-semibold
 
font-weight: var(--gcds-font-weights-semibold);
 
font-bold
 
font-weight: var(--gcds-font-weights-bold);

Exemples

Normal
font-regular

Cet élément utilise le poids de police par défaut.

<p class="font-regular">
  Cet élément utilise le poids de police par défaut.
</p>

Léger
font-light

Cet élément utilise un poids de police léger.

<p class="font-light">
  Cet élément utilise un poids de police léger.
</p>

Moyen
font-medium

Cet élément utilise un poids de police moyen.

<p class="font-medium">
  Cet élément utilise un poids de police moyen.
</p>

Semi-gras
font-semibold

Cet élément utilise un poids de police semi-gras.

<p class="font-semibold">
  Cet élément utilise un poids de police semi-gras.
</p>

Gras
font-bold

Cet élément utilise un poids de police gras.

<p>Cet élément utilise un poids de police <span class="font-bold"> gras</span>.</p>

Style conditionnel Appliquer ces styles à des tailles d’écran précises Utilisez un préfixe de mise en page réactive pour appliquer cette classe uniquement à une taille d’écran précise. Voici les points d’arrêt possibles :

  • xs: > 480px
  • sm: > 640px
  • md: > 768px
  • lg: > 1024px
  • xl: > 1280px
  
<div class="xs:font-medium ...">
  ...
</div>

  • focus
