## Button anatomy

<AnatomyList>
  <li>
    The <strong>container</strong> gives the button its visual prominence.
  </li>
  <li>
    The <strong>label</strong> states the action that'll happen when someone
    interacts with the button. Text is in sentence case, where only the first
    letter is capitalized.
  </li>
