## Problèmes résolus par les tableaux

Les tableaux aident à transformer des données brutes en informations claires, en organisant des éléments aux caractéristiques communes dans un format structuré.

Utilisez un tableau pour :

- Aider les personnes à parcourir et à comparer plusieurs éléments d’information, par exemple pour évaluer des indicateurs de performance ou un résumé financier. Ajoutez le tri de données pour leur permettre de réorganiser les données et de faire ressortir les tendances.
- Aider les personnes à trouver une information précise dans un ensemble de données plus vaste, par exemple pour rechercher un taux d’imposition ou l’adresse d’un bureau. Ajoutez le filtrage ou le tri pour leur permettre d’affiner les résultats.

**Remarque** : Sur mobile, la disposition adaptative du tableau enlève les colonnes. Le tableau se prête donc mieux aux tâches de recherche d’information plutôt qu’à la comparaison côte à côte. Si la comparaison de données est la tâche principale des utilisateurs, envisagez de réduire l’ensemble de données.
